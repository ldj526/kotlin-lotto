## 기능 목록
- 중복되지 않는 1~45 범위의 6개 수를 랜덤으로 생성한다.
- 얼마를 사는지 입력받는다.
  - 개수만큼 랜덤한 6개의 수를 생성한다. 
  - [예외처리] 1000으로 나누어떨어지지 않는다면 Error 메세지를 출력한다.
- 당첨 번호를 입력받는다.
  - 6개의 수는 쉼표로 구분해 입력받는다.
  - 보너스 번호도 입력받는다.
  - [예외처리] 1~45 사이의 수를 입력하지 않으면 Error 메세지를 출력한다.
- 입력받은 수와 비교하여 몇 개 맞는지 확인한다.
  - 6개의 번호와 비교한다.
  - 보너스 번호와 비교한다.
  - 맞은 개수에 따라 출력을 해준다.
- 수익률을 계산해 출력해준다.
